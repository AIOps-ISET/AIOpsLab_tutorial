# 文献阅读

## Word 模板

文献阅读需要做相关的笔记，在此我们仅仅推荐模板作为参考。

|      项目      |                             内容                             |
| :------------:| :----------------------------------------------------------: |
|    出版刊物    |                          期刊或会议                          |
|    研究对象    |                           研究目的                           |
|    文章类别    | 期刊-研究；期刊-综述；学位论文-博士；学位论文-硕士；会议-常规；会议-新概念；会议-工业驱动；会议-工具；会议-博士研究 |
|    划分类别    | 包括（应用领域，技术结合，基本概述，调研分析，算法优化，系统搭建） |
|    背景现状    |       现阶段背景存在的优势和不足（或本文工作的必要性）       |
|    研究工作    |                      针对现状所作的工作                      |
|    研究技术    |                  所用到的研究技术或具体工具                  |
|    应用领域    |                                                              |
|  研究思路总结  |                                                              |
|   创新性分析   |                                                              |
|  文章结构分析  |                                                              |
|    未来工作    |                                                              |
| 论文可借鉴内容 |                                                              |
|      建议      |                                                              |
